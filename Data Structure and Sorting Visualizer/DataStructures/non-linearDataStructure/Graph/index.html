How to Use This Tool

Creating and Editing Graphs

  • Click on the canvas to create a new node
  • Drag nodes to reposition them
  • Hold Alt and drag from one node to another to create an edge
  • Right-click on nodes or edges to delete them
  • Use the form controls to manually create nodes and edges
  • Load preset graphs using the preset buttons

Running the BFS Algorithm

  • Select a start node from the dropdown
  • Click "Run Algorithm" to start the visualization
  • Use the animation controls to play, pause, and step through the algorithm
  • Adjust the speed slider to control the animation speed
  • The queue shows which nodes are waiting to be processed
  • Node colors indicate their state:
    • White: Unvisited
    • Green: In Queue
    • Red: Visited
    • Orange: Currently Being Processed

About BFS Algorithm Visualization

This tool helps you visualize how the Breadth-First Search (BFS) algorithm works on graphs. BFS is a fundamental graph traversal algorithm that explores all vertices at the current depth before moving on to vertices at the next depth level.

How BFS Works

  1. Start at a selected node (the "source" node)
  2. Visit all adjacent nodes
  3. Mark the source node as visited
  4. For each adjacent node, if it hasn't been visited or queued yet, add it to the queue
  5. Move to the next node in the queue and repeat steps 2-4
  6. Continue until the queue is empty

Applications of BFS

  • Finding the shortest path in unweighted graphs
  • Web crawlers
  • Social networking
  • GPS Navigation systems
  • Network broadcasting

This visualization tool is inspired by the educational resources from the University of San Francisco.
