# Data Structures and Sorting Algorithms Visualization Tool

An interactive web-based visualization tool designed to bring fundamental computer science concepts to life. Through dynamic, real-time animations, this tool helps learners demystify abstract data structures and sorting mechanics by making their step-by-step execution completely tangible.

---

## 🚀 Features

### 1. Interactive Visualizations

* **Dynamic Animations:** Observe data shifting, pointers updating, and elements interacting in real-time.
* **Core Web Stack:** Smooth and lightweight rendering built completely using native **HTML, CSS, and JavaScript**, eliminating the need for heavy external frameworks.

### 2. Linear Data Structures Implemented

#### **Linked List**

* Full visual breakdown of a linear node-based structure.
* Animates key operations: **Node Creation**, **Deletion**, and **Traversal**.
* Visualizes nodes containing both **Data** (stored value) and **Next** (pointer reference).
* Interactive updates show exactly how pointer directions reassign dynamically during operations.

#### **Queue Implementations**

The tool provides two separate architectural variants for queues:

* **Queue Using Linked List:** A dynamic queue implementation mapping nodes linked via pointers using JavaScript classes. Animates `enqueue` and `dequeue` workflows seamlessly.
* **Queue Using Array:** A fixed or resizable array-based sequential structure visualization showcasing standard queue behavior.

---

## 🛠️ Built With

* **HTML5:** Structures the layout and visual frameworks for the animation panels.
* **CSS3:** Handles styling, transitions, and the concrete layout required for step-by-step element rendering.
* **JavaScript (ES6+):** Manages the core logical operations, dynamic data structures, state changes, and algorithmic execution.

## 💡 Why This Tool?

Understanding data structures like lists and queues can be incredibly challenging for beginners. By providing an interactive platform, this tool bridges the gap between static theory and dynamic code execution, ensuring optimized memory usage and better algorithm comprehension.
